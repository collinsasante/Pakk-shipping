## Airtable Automations for WhatsApp

### Setup WhatsApp Notifications via Airtable:

1. Go to **Automations** in your Airtable base
2. Create a new automation: **"When a record is updated" → Items table → Status field changes**
3. Add action: **Run a script** or **Send a webhook** to WhatsApp Business API

#### Option A — Airtable Script (Recommended)
```javascript
// Airtable Automation Script
const table = base.getTable('Items');
const record = await table.selectRecordAsync(input.config().recordId, {
  fields: ['CustomerName', 'OrderRef', 'Status', 'Customer']
});

const customerTable = base.getTable('Customers');
const customerId = record.getCellValue('Customer')[0].id;
const customer = await customerTable.selectRecordAsync(customerId, {
  fields: ['Phone', 'Name']
});

const phone = customer.getCellValueAsString('Phone').replace(/\D/g, '');
const name = customer.getCellValueAsString('Name');
const orderRef = record.getCellValueAsString('OrderRef') || 'N/A';
const status = record.getCellValueAsString('Status');

const message = `Hello ${name}, your package ${orderRef} is now *${status}*. Thank you for choosing Pakkmaxx! 📦`;

const response = await fetch(`https://graph.facebook.com/v18.0/${PHONE_NUMBER_ID}/messages`, {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${WHATSAPP_TOKEN}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    messaging_product: 'whatsapp',
    to: phone,
    type: 'text',
    text: { body: message }
  })
});

console.log('WhatsApp sent:', await response.json());
```

#### Option B — Third-party WhatsApp integration
Use services like **Twilio**, **360dialog**, or **WatiAPI** which provide Airtable-native integrations.
